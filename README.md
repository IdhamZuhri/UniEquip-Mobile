# 📱 UniEquip Mobile  
### University Equipment Inventory & Booking System

## 📌 Overview
UniEquip Mobile is an Android application designed to digitize and streamline the university equipment borrowing process.

Instead of manual tracking, this system provides a **real-time mobile platform** for students and administrators to manage equipment efficiently.

---

## 🚀 Features

### 👨‍🎓 Student Side
- Browse available equipment
- Reserve equipment in real-time
- View booking history
- Track request status

### 🛠️ Admin Side
- Manage equipment inventory
- Approve or reject booking requests
- Monitor usage and availability
- Generate reports

---

## 🧠 Problem Statement
Manual equipment booking systems often lead to:
- Double bookings
- Lack of transparency
- Inefficient tracking

---

## 💡 Solution
UniEquip Mobile solves this by:
- Providing a centralized digital system
- Enabling real-time updates via cloud database
- Improving booking accuracy and visibility

---

## 🛠️ Tech Stack

- **Language:** Java / Kotlin  
- **Platform:** Android  
- **Backend:** Firebase Cloud Firestore  
- **Tools:** Android Studio, Gradle
